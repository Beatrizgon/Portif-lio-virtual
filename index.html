Sweet Life

Sweet Life é um projeto acadêmico desenvolvido durante o segundo semestre da graduação, que tem como proposta a criação de uma loja virtual de sucos naturais. O site foi inspirado no modelo do Cosechas, uma plataforma que utilizo frequentemente para realizar compras, e busca oferecer uma experiência intuitiva e agradável aos usuários.

O projeto foi construído com HTML, CSS, JavaScript, PHP e MySQL, e apresenta funcionalidades como CRUD completo e um design limpo e responsivo. A estilização utiliza o framework Bootstrap, garantindo consistência visual e facilidade de navegação.

CareUp

CareUp é um projeto acadêmico focado em oferecer soluções acessíveis de seguros de vida e planos de saúde para jovens com idades entre 18 e 25 anos. A iniciativa tem como objetivo atrair um público que, muitas vezes, enfrenta dificuldades financeiras ou acredita ser invulnerável a problemas de saúde.

O projeto combina acessibilidade com uma navegação simples e intuitiva, proporcionando uma plataforma amigável que facilita o acesso a serviços essenciais. A interface foi projetada para ser atraente e funcional, pensando nas necessidades específicas dessa faixa etária.

Com uma abordagem inclusiva, CareUp conta com recursos que permitem a escolha entre diferentes tipos de seguros — individual ou em grupo —, organizados em três níveis: Básico, Top e Premium. O projeto utiliza tecnologias modernas, como HTML, CSS, JavaScript, PHP e MySQL, para criar um ambiente robusto e confiável.
